About Our Project


A data portal is a user interface which provides access to data that is resident and accessible via the portal. In this project we design and implement a data portal that is itself data driven. Data sources, their descriptions and APIs/interfaces to access the data will be input to the data portal. Our goal is when users choose the data portal, they will get choices of the different data sets that are available.

The adjective data-driven means that progress in an activity is compelled by data, rather than by intuition or by personal experience. In our project, to implement the data driven, we use PHP as the tool, MYSQL and XML as the data container. PHP can communicate between the data set and the web pages. MYSQL and XML can store data. Also, our project is now building on local server and will be put online soon.
